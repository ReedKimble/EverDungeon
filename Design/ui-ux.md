# UI / UX

The UX should be designed for old eyes and large screens. Fonts should be crisp and large enough to read comfortably. Icons should be large and distinguishable. A custom mouse cursor with particle and lighting effect options would be good as well.

The UI should be made of Window elements that can be sized and positioned by the user and the contents expands and flows as appropriate. Windows can be the Toolbar, Hotbars, Info Panels or Dialogs.

The Toolbar has 12 slots linked to keys 1 through =. The mouse scrollwheel selects between these tools/weapons and the mouse buttons activate modes of the selected tool.

Hotbars hold 8 shortcut slots for activated abilities and macros. Multiple hotbars are supported and the user can add and customize them as desired.

One hotbar will be active at a time linked to the keys Q E R T Z X C V and G/B will cycle through open hotbars.

The Input system uses Bindings to link input keys/mouse to named game-input elements which then are observed by game components.
